Overview of EPICS Version 4

What is EPICS Version 4?

Add structured data to EPICS:

  • Set of data types: pvData
  • New operations and network protocol to transport data: pvAccess
  • PVs now structures. Get, put monitor etc. individual fields
  • Atomic operations now possible
  • Channel RPC
  • Normative Types
  • RPC services

Applications:

  • Everything Version 3 does (but slightly better):
    • Better handling of strings, arrays and enums
    • Better support for operations with multiple record fields
  • Atomic group operations
  • Complex device control:
    • Distributed objects now possible
    • Monitorable and can understand RPC
  • Group operations or RPC
  • Experiment data:
    • Neutron data
    • areaDetector data
  • EPICS Version 4 services
